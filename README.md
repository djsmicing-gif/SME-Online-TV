# SME Online TV

This is a basic Android Studio project scaffold for "SME Online TV" using Kotlin and Jetpack Compose.

Package name: com.smeonlinetv.app

How to open:
1. Open Android Studio and choose "Open" -> select this repository root.
2. If you don't have the Gradle wrapper, you may need to install a compatible Gradle and Android Gradle Plugin.
3. Sync the project and run the app on an emulator or device.

Notes:
- This scaffold is minimal and omits the Gradle wrapper files. If you need a complete runnable build, run `gradle wrapper` locally or generate the project in Android Studio which will create the wrapper.
